# cybersecurity notes

## cia triad

- confidentiality → protect data from unauthorized access  
- integrity → ensure data is not altered  
- availability → ensure systems are accessible  

---

## key concepts

- authentication → verifying identity  
- authorization → granting access  
- least privilege → minimal access needed  
- defense in depth → multiple layers of security  
- attack surface → all possible entry points  

---

## common threats

- phishing → tricking users into giving sensitive information  
- malware → malicious software designed to harm systems  
- misconfiguration → insecure system setup  
- weak passwords → easily guessed credentials  

---

## basic security practices

- keep systems updated  
- use strong passwords  
- limit user permissions  
- monitor logs and activity  
- validate input  

---

## security mindset

- think like an attacker  
- understand how systems fail  
- focus on both prevention and detection
